    - 支援快速鍵

## 替代方案
- github page
  - 工具
    - [markmap.js](https://markmap.js.org/)

    - [demo](https://csongs.github.io/markmap-book)
  - 優點
    - 免費
    - 轉移方便
    - hugo 很多[theme](https://themes.gohugo.io/)
  - 缺點
    - 編輯麻煩
      - 寫一篇心智圖文件+hugo文章去嵌入
      - github action build 要時間 ,同步到page也要時間
    - 心智圖樣式少
    - 不能放圖片
